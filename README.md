# 技术栈
[预览](https://45.76.222.218:8000)
## 前端
> mpvue + iview + vue-router + vuex + echarts
## 后端
> spring + springmvc + hibernate + mysql + redis

## 项目结构
```
├── titan_core                  // 封装核心框架
├── titan_crawler               // 网页爬虫
├── titan_monitor               // 系统监控
├── titan_quartz                // 定时任务
├── titan_routine               // 微信小程序
├── titan_server                // 接口服务
└── titan_web                   // 后台页面
```
